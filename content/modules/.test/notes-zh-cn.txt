## 5. 代码

行内代码：使用 `print("你好")` 即可

缩进式代码块：

    def say_hello():
        print("Hello Markdown!")

围栏式代码块（这里故意不写 ``` 语言标识，让渲染器自己判断）：

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)
```

```javascript
const greet = name => `你好，${name}！`;
console.log(greet("世界"));
```
